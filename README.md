# Toggle Spec

Jump to the spec/source counter-part of the current file.

![](example/run.gif)

## Usage

### Manual

Press `Ctrl+Shift+P` (`Cmd+Shift+P` on a Mac) to open the Command Pallete and type `Toggle Spec`.

### Keyboard Shortcut

Bind `extension.toggleSpec` to a keybinding.

![](example/bind.gif)
